对于对象可以使用 递归来响应式化，但是数组我们也需要处理

- push
- pop
- shift
- unshift
- reverse
- sort
- splice

需要怎么做？

1. 在改变数组的数据时，要发出通知
  - vue2 中的缺陷，数组发生变化，设置length 没法通知（vue3 中使用了 Proxy 语法 ES6语法 解决）
2. 加入的元素变成响应式


技巧：如果一个函数已经定义了，但是我们需要扩展其功能，一般处理办法

1. 使用一个临时的函数名存储函数
2. 重新定义原来的的函数
3. 定义扩展的功能
4. 调用临时的那个函数
